# Learner Progress Dashboard - Coding Challenge

## Getting Started 

## Author : MSIZI SAMUEL MZOBE

1. Run `composer install`
2. Download `xampp` from `https://www.apachefriends.org/download.html`'
3. Make sure it's `PHP.8^`
4. Run xampp
6. Configure your `.env` file from the example
7. Generate the App Key: `php artisan key:generate`
8. Run migrations and seeders: `php artisan migrate --seed`
9. Start the development server: `php artisan serve`
